# 2048
A simple CLI-based 2048 game written in C++ for Linux. You can build 2048.cpp as

## Build and Run
```bash
g++ 2048.cpp -std=c++17 -Wall -Wextra -pedantic -o 2048.out
```

and run 2048.out as `./2048.out`.

## Controls
  Realtime move keys:
    W / K / 8 : Up
    A / H / 4 : Left
    S / J / 2 : Down
    D / L / 6 : Right
    Q : Quit game
  Command mode:
    :help  Show this help
    :quit  Exit game
